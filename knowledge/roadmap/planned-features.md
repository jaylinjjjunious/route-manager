# Planned Features

These are Phase 3 recommendations from `docs/CURRENT_STATE.md`. None have been started.

## Server-Synced Data
- Move all data (not just shower proofs) to server-side storage.
- Replace localStorage as the primary data store for jobs, routes, habits, and settings.

## Multi-Day Route Planning
- Plan routes across multiple days.
- Carry forward incomplete jobs to the next day's route.

## Team/Manager Dashboard
- Allow a manager to view and assign jobs to workers.
- Requires multi-user support and role-based access control.

## Real-Time GPS Tracking
- Track worker location during Ride Mode.
- Enable live route updates and ETA calculations.

## Customer-Facing Proof Delivery
- Share proof of completion directly with customers.
- Generate shareable links or PDF reports.

## Automated Billing/Payment Reconciliation
- Link completed jobs to billing.
- Automatic payment tracking and reconciliation.

## Status

All planned features are in the backlog stage. No implementation work has begun.
